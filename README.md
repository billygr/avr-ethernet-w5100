# avr-ethernet-w5100

Wiznet 5100 library for AVR ATMega Devices

## Features
  * Arduino Hardware Compatible (tested with Arduino Uno and Arduino Ethernet Shield)
  * Small code size (Arduino 1.6.5 Webserver example uses 12576 bytes of program storage space, 570 bytes memory). avr-ethernet-w5100 Hello World is around 3400 bytes code/375 bytes RAM

## Requirements
  * avr-gcc (tested with avr-gcc 4.9.1)

## Howto
Just run make. The resulting code works directly with Arduino and Arduino Ethernet Shield. You can upload it with avrdude or with Xloader
